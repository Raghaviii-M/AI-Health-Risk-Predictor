# AI Health Risk Predictor

AI Health Risk Predictor – A machine learning system that analyzes patient health parameters to predict potential abnormal medical test outcomes. Uses healthcare data such as age, gender, medical condition, admission type, and billing details to classify results as Normal, Abnormal, or Inconclusive. Provides predictions, confidence scores, health suggestions, and personalized risk visualizations through an interactive Streamlit dashboard, supporting preventive healthcare and better medical decision-making.

## Technologies
Python, Scikit-learn, Streamlit, Pandas

## Features
Health risk prediction

Confidence score

Personal health dashboard

Trend visualization
