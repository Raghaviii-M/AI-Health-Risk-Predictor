# **AI-Health-Risk-Predictor**
The AI-Health-Risk-Predictor is a machine learning system that analyzes patient data to predict medical test outcomes as Normal, Abnormal, or Inconclusive. It includes a Streamlit dashboard that provides predictions, confidence scores, and health insights to support early healthcare decisions.

## SDG Goal
Good Health and Well-Being (SDG 3)

## Technologies Used
Python,
Scikit-learn,
Streamlit,
Pandas


## Features
1. Health risk prediction
2. Confidence score
3. Personal health dashboard
4. Trend visualization
